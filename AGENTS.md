# KiloNode Agent Rules

KiloNode is an independent ISC-licensed packet-radio node and BBS project.

Do not copy code, comments, structure, constants, tables, or parser behavior
from LinBPQ, BPQ32, or any GPL source.

Use only:

- Public protocol specifications
- Independently written notes
- Black-box tests against running systems
- Project-owned tests and packet captures
- Permissively licensed references

Code rules:

- Use C17.
- Use tabs for indentation in code.
- Use strict bounds checks.
- Avoid unsafe string functions.
- Do not add GPL dependencies.
- Do not use `sudo`.
