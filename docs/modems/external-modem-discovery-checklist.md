# External Modem Discovery Checklist

Complete this checklist before implementing an external modem adapter.

- Project or binary location.
- Licence and redistribution limits.
- Supported operating modes.
- Control interface.
- Data interface.
- Framing and message boundaries.
- Status query behavior.
- Receive data path.
- Transmit data path.
- PTT or radio-control ownership.
- Test mode or non-transmitting mode.
- Example receive capture.
- Failure modes and disconnect behavior.
- Required operator controls.
- No source vendoring requirement confirmed.

Passing this checklist does not enable transmit. It only makes a future adapter
implementation possible.
