# Clean-Room Policy

KiloNode is an independent ISC-licensed implementation. The project must not
copy source code, comments, internal structure, tables, constants, or
implementation-specific parser behavior from LinBPQ, BPQ32, or any GPL source.

Allowed inputs:

- Public protocol specifications
- Packet captures generated by project-owned tests
- Black-box behavior from running nodes
- Independently written notes
- Permissively licensed references
- Project-owned tests

Disallowed inputs:

- Copied GPL source
- Translated GPL source
- Comments or function layouts derived from GPL source
- Generated code from GPL implementations
- Parser behavior taken from source-code reading

Compatibility work must be based on public behavior, protocol documents, and
tests written for this project.
