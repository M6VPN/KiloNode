# M2.7 Hobbyist Preview Release Polish

M2.7 focuses on making the current KiloNode work easy to try as a hobbyist
preview. It favors clear commands and a small no-hardware smoke path over more
protocol scaffolding.

## Goal

Provide a practical local preview for a future `v0.2-alpha` tag:

- build KiloNode
- run one no-hardware smoke script
- validate the hobbyist local config
- demonstrate AX.25 loopback payload exchange
- demonstrate CONNECT dry-run planning
- confirm real transmit paths stay disabled

## Primary User Path

```sh
./scripts/build.sh
./scripts/hobbyist-smoke.sh
./build/kilonoded --config packaging/examples/kilonode-hobbyist-local.conf --check-config
```

Optional AX.25 demos:

```sh
./build/kilonode-compat run-ax25-loopback tests/fixtures/ax25-loopback/connect-i-rr-disconnect.loop
./build/kilonode-compat run-ax25-connect-dry-run tests/fixtures/ax25-connect-dry-run/basic.connectdry
```

## Included

- Local daemon config for hobbyist preview.
- Local control socket and local TCP shell config.
- Local BBS store config.
- No-hardware AX.25 loopback and CONNECT dry-run demos.
- No-transmit check in the smoke path.

## Still Blocked

- Live CONNECT.
- RF BBS access.
- Real TX queue writes.
- TX dispatch.
- NET/ROM.
- BBS forwarding.
- FX.25 FEC or wrapping.

## Checks

