# M2.9 Hobbyist v0.2-alpha Readiness

M2.9 makes the current M2 work easier to try as a hobbyist preview. It adds a
first-run path, product docs, safe example configs, Mercury OFDM discovery
material, and scripts that check readiness without hardware.

Included checks:

- Build and deterministic tests.
- Hobbyist smoke check.
- No-transmit config check.
- External modem config check.
- Mercury discovery check.
- Packaging docs and example config check.
- M2 readiness check.

Blocked features remain blocked:

- Live CONNECT.
- Real TX.
- TX dispatch.
- RF BBS access.
- NET/ROM.
- Forwarding.
- Mercury, VARA, and ARDOP protocol implementations.
- FX.25 FEC.

Tag recommendation criteria:

- `./scripts/v02-alpha-readiness-check.sh` passes.
- `./scripts/ax25-no-transmit-check.sh` passes.
- New product and Mercury docs are present.
- Example configs keep TX, CONNECT, auto-start, and dispatch disabled.

If those pass, the tree is a candidate for a v0.2-alpha hobbyist preview tag.
