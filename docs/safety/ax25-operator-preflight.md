# AX.25 Operator Preflight

This preflight is for future real TX only. It is not used in automated tests.

- [ ] Operator has legal authority and licence for the station and band.
- [ ] Callsign is configured and correct.
- [ ] Frequency and band plan are documented.
- [ ] Power level is set and logged.
- [ ] Antenna, dummy load, or isolated test setup is selected.
- [ ] TX port is identified and explicitly enabled.
- [ ] PTT path is understood and tested in isolation.
- [ ] Emergency stop path is available.
- [ ] Logs are enabled and retained.
- [ ] BBS access is not enabled unless separately approved.
- [ ] NET/ROM is not enabled unless separately approved.
- [ ] FX.25 wrapping is not enabled unless separately approved.
