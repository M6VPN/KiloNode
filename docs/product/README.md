# KiloNode Product Notes

This section keeps the hobbyist preview path in one place. It is for people who
want to build KiloNode, run safe local checks, inspect modem status, and try the
receive-only or simulator features without needing RF hardware.

Start here:

- [Hobbyist v0.2-alpha preview](hobbyist-v0.2-alpha-preview.md)
- [First-run checklist](first-run-checklist.md)
- [What works today](what-works-today.md)
- [What is still blocked](what-is-still-blocked.md)
- [Modem support status](modem-support-status.md)

The current preview does not enable real TX, CONNECT, RF BBS access, NET/ROM,
forwarding, or FX.25 FEC.
