# What Is Still Blocked

These items remain blocked or planned for later milestones:

- Live CONNECT.
- Real connected-mode RF sessions.
- Real TX queue writes.
- TX dispatch.
- RF BBS access.
- NET/ROM.
- BBS forwarding and FBB forwarding.
- LinBPQ command compatibility.
- Mercury adapter implementation.
- VARA adapters.
- ARDOP adapter.
- FX.25 FEC and wrapping.
- PTT policy.
- Channel busy policy.

The v0.2-alpha readiness path does not remove any of these blocks.
