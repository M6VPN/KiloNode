# KiloNode Reference Index

This directory stores protocol references used by KiloNode planning and
implementation. Reference files are not edited in normal development.

| File             | Area  | Purpose                                      | Standing    | Used in M1.32 | Notes |
|------------------|-------|----------------------------------------------|-------------|---------------|-------|
| AX25.2.2.pdf     | AX.25 | Link layer frame, control, timer, and state material for AX.25 version 2.2. | Normative for AX.25 planning. | Yes | Includes frame structure, address fields, control fields, procedures, timers, parameters, and SDL appendices. |
| FX-25_01_06.pdf  | FX.25 | Draft FEC wrapper model for carrying AX.25 packets with compatibility goals. | Informative draft for FX.25 planning. | Yes | Describes an FX.25 layer around AX.25 packets, correlation tags, FEC code blocks, padding, and fallback considerations. |
