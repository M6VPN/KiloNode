# v0.2-alpha Checklist

`v0.2-alpha` should be a hobbyist preview release. It should be easy to build,
easy to try locally, and honest about what is still blocked.

## Required Checks

- [ ] `./scripts/build.sh`
- [ ] `./scripts/hobbyist-smoke.sh`
- [ ] `./scripts/ax25-no-transmit-check.sh`
- [ ] `./scripts/m2-readiness-check.sh`
- [ ] `./scripts/test.sh`
- [ ] `./build/kilonoded --config packaging/examples/kilonode-hobbyist-local.conf --check-config`

## Release Notes Must Say

- KiloNode supports local diagnostics, local BBS storage, and AX.25 simulator
  demos.
- Dire Wolf TCP KISS is the first practical receive-only modem path.
- VARA FM/HF, Mercury OFDM, and ARDOP are planned external modem adapters, not
  current working integrations.
- AX.25 loopback and CONNECT dry-run are local proof tools, not live RF
  connected-mode support.
- Real TX queue writes and dispatch remain disabled.
- RF BBS, NET/ROM, BBS forwarding, and FX.25 remain planned.

## Manual Spot Checks

- Run `./scripts/hobbyist-smoke.sh` from a clean build tree.
- Read `docs/getting-started-hobbyist.md` and confirm the commands match the
  current binaries.
- Check that example configs bind only to localhost unless clearly labelled as
  future lab work.
- Check that the no-transmit script covers every non-lab example config.
